# User Context - Flappy Kiro Game

## Project Details
- **Game Type**: Flappy Kiro (Flappy Bird style)
- **Technology Stack**: Vanilla JavaScript, HTML5 Canvas
- **Character Sprite**: kiro-logo.png

## Game Specifications
### Physics
- Gravity: 0.3
- Jump Power: -6

### Obstacles
- Movement Speed: 1.5 pixels per frame
- Spawn Rate: Every 180 frames

### Game States
- start: Initial screen with instructions
- playing: Active gameplay
- gameOver: End screen with restart option

### Controls
- SPACE key or click to start/jump
- Restart functionality on game over

## User Preferences
- Wants balanced, responsive physics
- Prefers simple, playable implementation
- Wants clear start screen before gameplay begins
